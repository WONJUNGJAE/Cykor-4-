# CykorProject



## 주요 기능

- **CRUD 게시판**
  - 게시글 목록 조회
  - 게시글 작성
  - 게시글 수정
  - 게시글 삭제
- **회원가입**
  - 아이디, 비밀번호로 회원가입 가능
- **로그인**
  - 로그인 시 JWT 토큰 발급
  - 로그인 상태에서만 글 작성, 수정, 삭제 가능

---

## 사용 방법

1. **프로젝트 실행**
   - 백엔드가 5000번, 프론트엔드가 3000번 포트에서 실행된다.
2. **브라우저에서 접속**
   - `http://localhost:3000/` 주소로 접속.
3. **회원가입/로그인**
   - 로그인 상태가 아니면 회원가입 또는 로그인 화면이 나온다.
   - 로그인하면 게시판 화면으로 이동한다.
4. **게시판 사용**
   - 게시글 작성, 수정, 삭제가 가능하다.
